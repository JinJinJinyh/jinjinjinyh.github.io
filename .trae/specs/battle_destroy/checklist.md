
# 战地破坏 - 验证清单

## 项目设置
- [ ] Unreal Engine 5项目可以正常启动
- [ ] Chaos Physics已启用
- [ ] 基本输入映射正常工作（移动、跳跃、射击）

## 方块系统
- [ ] 方块可以放置在世界中
- [ ] 方块有生命值系统（0-60）
- [ ] 方块在不同生命值阶段显示不同贴图（完好→轻微损坏→严重损坏→摧毁）
- [ ] 支持多种材质的方块（木头、石头、混凝土、金属）

## 武器系统
- [ ] 突击步枪可以正常射击
- [ ] 突击步枪每发子弹对方块造成1点伤害
- [ ] 连续射击60次可以摧毁一个方块
- [ ] RPG可以正常发射
- [ ] RPG可以造成3x3x3范围的方块破坏

## 物理与结构稳定性
- [ ] 破坏承重墙后，建筑的剩余部分会倒塌
- [ ] 倒塌的方块会变成物理碎片
- [ ] 碎片有真实的物理模拟（重力、碰撞）
- [ ] 碎片会在一段时间后自动清理

## 整体游戏体验
- [ ] 玩家可以在地图中自由移动
- [ ] 游戏可以以至少30 FPS运行（中等配置）
- [ ] 所有系统整合正常工作
- [ ] 游戏玩法直观有趣
